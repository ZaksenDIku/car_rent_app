# Car Rent App

 **Deployet version:**  
https://car-rent-app-akbfh5bwdteuduhg.switzerlandnorth-01.azurewebsites.net/

---

## Projektbeskrivelse
Kort beskrivelse af applikationen.  
(Fx: En simpel webapplikation til administration og booking af biler.)

---

## Funktionalitet (kort)
- Oprette og vise biler  
- Oprette og administrere bookinger  
- Simpelt web-interface (Thymeleaf)  
- Database (MySQL)

---

## Teknologi
- Java 21  
- Spring Boot  
- Thymeleaf  
- MySQL  
- Maven  
---

## Installation (lokalt)
```bash
git clone <repo-url>
cd car_rent_app
mvn spring-boot:run
---
